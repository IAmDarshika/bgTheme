Example to demonstate onKey Event on Keyboard

Press "q" or "Q" key ANYWHERE INSIDE PAGE to set a red background color.

Press "w" or "W" key ANYWHERE INSIDE PAGE to set a green background color.

Press "e" or "E" key ANYWHERE INSIDE PAGE to set a blue background color.

Press "r" or "R" key ANYWHERE INSIDE PAGE to set a yellow background color.

Press "t" or "T" key ANYWHERE INSIDE PAGE to set a pink background color.

Press "y" or "Y" key ANYWHERE INSIDE PAGE to set a light yellow background color.
